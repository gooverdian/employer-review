# Должность


**URL:** /position

**POST:**
Принимает на вход тело запроса:
```json
{
	"name":"lawyer"
}
```

В случае успешного сохранения возвращается:
```json
{
	"id":4,
	"name":"lawyer"
}
```

В противном случае 
```json
{
	"id":"null",
	"name":"null"
}
```

**GET:**
Принимает необязательный параметр text

Возвращает результат в следующем виде:
```json
[
	{"id":0,"name":"Gardener"},
	{"id":1,"name":"Senior Gardener"}
]
```

В случае пустого ответа:
```json
[]
```
